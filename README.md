# 📊 Sales Data Analysis using Python & Web Scraping

## 📌 Project Overview
This project demonstrates a complete data analysis pipeline, starting from data collection using web scraping (Selenium) to extracting meaningful business insights using Python.

## 🛠 Tools & Technologies
- Python  
- Pandas  
- Selenium (Web Scraping)  
- Matplotlib & Seaborn  
- Jupyter Notebook  

## 📊 Project Description
In this project, I collected sales data from multiple web sources using Selenium and analyzed eight different datasets. The goal was to understand key business factors affecting sales performance.

## ❓ Key Questions
- How do prices affect product sales?  
- What is the seasonal impact on product demand?  
- How do promotional campaigns influence revenue?  
- What is the relationship between number of orders and total revenue?  
- How can distribution strategies be improved based on data insights?  

## 📈 Key Insights
- High-priced products contribute significantly to revenue despite lower sales volume.  
- Strong seasonal trends were observed, with increased demand during winter.  
- Inventory management plays a key role in meeting peak demand.  
- Promotional campaigns have a noticeable impact on revenue growth.
